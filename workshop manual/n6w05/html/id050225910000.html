2014 - Mazda6 - Transmission/Transaxle

DTC U0073:00, U0100:00, U0121:00, U0131:00, U0155:00 [FW6A-EL]



DTC U0100:00

Communication error to PCM (HS CAN)

DETECTION CONDITION

  • Under the following condition, the TCM cannot receive the signal from PCM (HS CAN):

    • Battery voltage is 10 V or more.

    Diagnostic support note

  • The check engine light illuminates if the TCM detects the above malfunction condition during the first drive cycle.

  • The automatic transaxle warning light does not illuminate.

  • PENDING CODE is available.

  • FREEZE FRAME DATA is available.

  • DTC is stored in the TCM memory.

FAIL-SAFE FUNCTION

  • Inhibits learning control.

  • Inhibits neutral idle control.

  • Inhibits AAS.

POSSIBLE CAUSE

  • CAN communication line malfunction between PCM and TCM (HS CAN)

  • TCM malfunction

SYSTEM WIRING DIAGRAM

Not applicable
