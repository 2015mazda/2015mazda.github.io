2014 - Mazda6 - Transmission/Transaxle

TCM CONFIGURATION [FW6A-EL]

NOTE:

  • The TCM is built into the control valve body.

1. Verify TCM configuration implementation necessity for replacement parts.

Replacement part

Configuration necessity

Control valve body replacement

Necessary

Oil pressure switch replacement

Unnecessary

Automatic transaxle replacement

Unnecessary

ON/OFF solenoid replacement

Unnecessary

Coupler component replacement

Unnecessary

2. Connect the M-MDS to the DLC-2.
3. After vehicle identification, the following can be selected from the M-MDS initialization screen.

  • Select “Module Programming”

4. Then, select the following from the screen menu.

  • Select “Programmable Module Installation”

  • Select “TCM”

5. Perform the configuration following the instructions on the screen.
6. Verify that no DTCs are stored. (See ON-BOARD DIAGNOSTIC SYSTEM DTC INSPECTION [FW6A-EL].)

NOTE:

  • If any DTC (U code) is output, the system is normal.

7. Delete the U code and verify that no DTC is output. (See ON-BOARD DIAGNOSTIC SYSTEM DTC INSPECTION [FW6A-EL].)

  • If any DTC other than U code is output, repair the malfunctioning location according to the applicable DTC troubleshooting. (See ON-BOARD DIAGNOSTIC SYSTEM DTC TABLE [FW6A-EL].)
