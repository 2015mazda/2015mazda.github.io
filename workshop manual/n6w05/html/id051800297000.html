2014 - Mazda6 - Transmission/Transaxle

SELECTOR LEVER INSPECTION

NOTE:

    There are two types of automatic transaxle shift mechanisms.

  • Type A: Gate-shaped key type. There is no lock release button on the selector lever knob.

  • Type B: Gate-shaped straight type. There is a lock release button on the selector lever knob.

1. Switch the ignition ON (engine off or on).
2. Perform the following procedures to inspect the selector lever.

  • If there is any malfunction, adjust the selector cable. (See AUTOMATIC TRANSAXLE SHIFT MECHANISM REMOVAL/INSTALLATION.)

a. Verify that the selector lever can be operated as shown in the figure.
