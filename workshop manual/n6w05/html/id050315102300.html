2014 - Mazda6 - Transmission/Transaxle

NOISE OCCURS AT IDLE WHEN VEHICLE IS STOPPED IN ALL POSITIONS [FW6A-EL]



TROUBLESHOOTING ITEM

Noise occurs at idle when vehicle is stopped in all positions

DESCRIPTION

  • Transaxle is noisy in all positions when vehicle idling.

POSSIBLE CAUSE

  • Engine control system malfunction

  • Automatic transaxle internal parts malfunction

    • Oil pump malfunction

    • Torque converter internal parts malfunction

Diagnostic procedure

STEP

INSPECTION

ACTION

1

 VERIFY IF MALFUNCTION CAUSE IS ENGINE CONTROL SYSTEM OR TORQUE CONVERTER

  • Inspect the engine condition.

  • Is there any malfunction?

Yes

Go to the appropriate symptom troubleshooting.

(See SYMPTOM DIAGNOSTIC INDEX [SKYACTIV-G 2.5].)

No

Replace the automatic transaxle.

(See AUTOMATIC TRANSAXLE REMOVAL/INSTALLATION [FW6A-EL].)
