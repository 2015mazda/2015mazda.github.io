2014 - Mazda6 - Transmission/Transaxle

TRANSAXLE OVERHEATS [FW6A-EL]



TROUBLESHOOTING ITEM

Transaxle overheats

DESCRIPTION

  • Burnt smell emitted from the transaxle.

  • Smoke is emitted from the transaxle.

POSSIBLE CAUSE

  • Signal malfunction

    • APP sensor malfunction

    • Brake switch malfunction

    • CKP sensor malfunction

    • ECT sensor No.1 malfunction

  • ATF malfunction

  • Oil cooler malfunction

  • Control valve body malfunction

CAUTION:

  • Verify the malfunction symptom according to not only the PID value but also the symptom troubleshooting.

Diagnostic procedure

STEP

INSPECTION

ACTION

1

 INSPECT SIGNAL PARTS FOR MALFUNCTION

  • Inspect the value at the following PCM PIDs using the M-MDS.

    (See ON-BOARD DIAGNOSTIC TEST [SKYACTIV-G 2.5].)

    • APP (APP sensor)

    • BOO (Brake switch)

    • ECT (ECT sensor No.1)

    • RPM (CKP sensor)

  • Is there any malfunction?

Yes

Repair or replace any malfunctioning parts according to the inspection result.

No

Go to the next step.

2

 INSPECT ATF LEVEL

  • Inspect the ATF level.

    (See AUTOMATIC TRANSAXLE FLUID (ATF) INSPECTION [FW6A-EL].)

  • Is there any malfunction?

Yes

Adjust the ATF level to the specification.

(See AUTOMATIC TRANSAXLE FLUID (ATF) REPLACEMENT [FW6A-EL].)

No

Go to the next step.

3

 VISUALLY INSPECT OIL COOLER

  • Visually inspect the oil cooler.

  • Is there any malfunction?

Yes

Repair or replace any malfunctioning parts according to the inspection result.

(See OIL COOLER REMOVAL/INSTALLATION [FW6A-EL].)

No

Go to the next step.

4

 VERIFY IF MALFUNCTION CAUSE IS CONTROL VALVE BODY

  • Inspect the value at the following TCM PIDs using the M-MDS.

    (See ON-BOARD DIAGNOSTIC SYSTEM PID/DATA MONITOR INSPECTION [FW6A-EL].)

    • ECU_A

    • ECU_B

    • ECU_C

    • GEAR_SEL

    • LINEDES

    • OP_SW1

    • OP_SW2

    • OP_SW3

    • OP_SW4

    • OSS

    • SS1

    • SS2

    • SS3

    • SS4

    • SSLU

    • SSLU_PRES

    • SSP

    • SSP_PRES

    • TR

    • TSS

    • VSS

  • Is there any malfunction?

Yes

Replace the control valve body, then go to the next step.

(See CONTROL VALVE BODY REMOVAL/INSTALLATION [FW6A-EL].)

No

Symptom troubleshooting is completed.
