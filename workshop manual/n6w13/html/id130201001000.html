2014 - Mazda6 - I-ELOOP

DTC U0001:88, U0100:00, U0155:00 [i-ELOOP]



DTC U0001:88

CAN system communication error (HS-CAN)

DETECTION CONDITION

  • Malfunction in CAN bus communication line.

FAIL-SAFE FUNCTION

  • Not applicable

POSSIBLE CAUSE

  • CAN bus communication line malfunction

  • DC-DC converter malfunction

SYSTEM WIRING DIAGRAM

  • Not applicable



DTC U0100:00

CAN communication: communication error to PCM

DETECTION CONDITION

  • Communication error between the DC-DC converter and PCM continues for 5 s or more.

FAIL-SAFE FUNCTION

  • Not applicable

POSSIBLE CAUSE

  • CAN communication line between DC-DC converter and PCM malfunction

  • PCM malfunction

  • DC-DC converter malfunction

SYSTEM WIRING DIAGRAM

  • Not applicable



DTC U0155:00

CAN communication: communication error to instrument cluster

DETECTION CONDITION

  • Communication error between the DC-DC converter and instrument cluster continues for 5 s or more.

FAIL-SAFE FUNCTION

  • Not applicable

POSSIBLE CAUSE

  • CAN communication line between DC-DC converter and instrument cluster malfunction

  • Instrument cluster malfunction

  • DC-DC converter malfunction

SYSTEM WIRING DIAGRAM

  • Not applicable

Diagnostic Procedure

  • Inspect according to the diagnostic procedure in ON-BOARD DIAGNOSTIC [MULTIPLEX COMMUNICATION SYSTEM]. (See FOREWORD.)
