2014 - Mazda6 - I-ELOOP

No.3 i-ELOOP-RELATED GAUGE IN LCD DOES NOT MOVE [i-ELOOP]

Description

  • Only three gauge segments of the regenerative braking power generation gauge are displayed/only one gauge segment of the capacitor accumulation gauge is displayed

Possible Cause

Bypass Mode Is Operating

  • Vehicle electric load (current consumption) is 50 A or more

  • Voltage reduction circuit in DC-DC converter is malfunctioning

  • Ambient temperature is less than 0 °C {32 °F}

  • Battery charge condition is less than 65 %

Instrument cluster incorrect display

  • Instrument cluster malfunction

Other

  • Engine-related part malfunction

  • Transmission-related part malfunction

Step

Results

 VERIFY DTCs

    (See ON-BOARD DIAGNOSTIC TEST [SKYACTIV-G 2.5])

    (See ON-BOARD DIAGNOSTIC SYSTEM DTC INSPECTION [FW6A-EL])

    (See DTC INSPECTION [INSTRUMENT CLUSTER])

 VERIFY MALFUNCTION CONDITION

 ASK CUSTOMER

5

 VERIFY CAUSE OF BYPASS MODE OPERATION

  • Display PID DCDC_OUT_I using the M-MDS data logger function.

  • Is “DCDC_OUT_I“ value 50 A or more?

Yes

Explain to the customer that the vehicle is normal and that i-ELOOP does not operate because of the vehicle electrical load.

No

Go to the next step.

6

 VERIFY AMBIENT TEMPERATURE

  • Display PID AMB_TEMP using the M-MDS data logger function.

  • Is “AMB_TEMP” value less than 0 °C {32 °F}?

Yes

Explain to the customer that the vehicle is normal and that i-ELOOP does not operate to assure battery power supply.

No

Go to the next step.

7

 VERIFY BATTERY CHARGE CONDITION

  • Display PID BATT_SOC using the M-MDS data logger function.

  • Is “BATT_SOC” value less than 65%?

Yes

Explain to the customer that the vehicle is normal and that i-ELOOP does not operate to prioritize battery charging.

No
