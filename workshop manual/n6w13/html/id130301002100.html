2014 - Mazda6 - I-ELOOP

No.1 "i-ELOOP Inspection Required" IS DISPLAYED IN LCD [i-ELOOP]

Description

  • “i-ELOOP Inspection Required” is displayed in LCD

  • Master warning light is illuminated

Possible Cause

i-ELOOP-related component malfunction (DTCs are detected)

  • Capacitor (i-ELOOP) malfunction

  • DC-DC converter malfunction

  • PCM malfunction

  • i-ELOOP-related part wiring harness malfunction

Instrument cluster incorrect display

  • Instrument cluster malfunction

Diagnostic Procedure



Step

Inspection

Results

Action

1

 VERIFY DTCs

  • Using the M-MDS, display the DTCs related to the PCM, DC-DC converter, and instrument cluster.

  • Are any DTCs displayed?

Yes

Repair the malfunctioning part according to the applicable DTC troubleshooting.

No

Inspect the instrument cluster and replace it if there is any malfunction.

(See INSTRUMENT CLUSTER REMOVAL/INSTALLATION)
