1. Remove in the order indicated in the table.

Bolt Removal Note

1. Remove the rear caliper installation bolt shown in the figure and lift up the rear caliper.

Bolt Installation Note

1. Clean the exposed area of the piston.
2. Rotate the piston clockwise slowly using the SST and push the piston completely until the piston grooves are in the position shown in the figure.

3. Return the rear caliper to the original position and install the rear caliper installation bolt.
