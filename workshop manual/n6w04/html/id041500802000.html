2014 - Mazda6 - Brakes

DSC OFF SWITCH INSPECTION

NOTE:

  • The DSC OFF switch is integrated with the cluster switch.

  • For the DSC OFF switch removal/installation, refer to CLUSTER SWITCH REMOVAL/INSTALLATION. (See CLUSTER SWITCH REMOVAL/INSTALLATION.)

1. Remove the cluster switch. (See CLUSTER SWITCH REMOVAL/INSTALLATION.)
2. Verify that the resistance between cluster switches B and C is as indicated in the table.

  • If the resistance cannot be verified as indicated in the table, replace the cluster switch.

3. Apply battery positive voltage to cluster switch terminal J, and connect terminal I to ground.

4. Verify that the LED illuminates.

  • If there is any malfunction, replace the cluster switch.
