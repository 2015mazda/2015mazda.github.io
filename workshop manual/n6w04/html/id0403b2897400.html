2014 - Mazda6 - Brakes

NO.8 MASTER WARNING LIGHT REMAINS ILLUMINATED AND A MESSAGE "SCBS Inspection Required" IS INDICATED IN THE DISPLAY [DYNAMIC STABILITY CONTROL (DSC)]



8

Master warning light remains illuminated and a message “SCBS Inspection Required” is indicated in the display*

POSSIBLE CAUSE

  • Laser sensor detects a malfunction.

  • DSC HU/CM internal malfunction.

  • DSC HU/CM detected malfunction (input and output device malfunction).

  • PCM and/or instrument cluster detect a malfunction.

*
Vehicle with TFT LCD display.
Diagnostic procedure

STEP

INSPECTION

ACTION

1

 CONFIRM LASER SENSOR DTC

  • Retrieve the DSC HU/CM and laser sensor DTC using the M-MDS.

    (See ON-BOARD DIAGNOSIS [LASER SENSOR].)

  • Are any DTCs present?

Yes

Go to the applicable DTC inspection.

(See ON-BOARD DIAGNOSIS [LASER SENSOR].)

No

Go to the next step.

2

 CONFIRM DSC HU/CM DTC

  • Retrieve the DSC HU/CM and laser sensor DTC using the M-MDS.

    (See ON-BOARD DIAGNOSIS [DYNAMIC STABILITY CONTROL (DSC)].)

  • Are any DTCs present?

Yes

Go to the applicable DTC inspection.

(See ON-BOARD DIAGNOSIS [DYNAMIC STABILITY CONTROL (DSC)].)

No

Go to the next step.

3

 CONFIRM PCM DTC

  • Retrieve the PCM DTC using the M-MDS.

    (See ON-BOARD DIAGNOSTIC TEST [SKYACTIV-G 2.5].)

  • Are any DTCs present?

Yes

Go to the applicable DTC inspection.

(See DTC TABLE [SKYACTIV-G 2.5].)

No

Retrieve the instrument cluster DTC using the M-MDS.

(See DTC INSPECTION [INSTRUMENT CLUSTER].)

  • If any DTCs are present, go to the applicable DTC inspection.

    (See DTC TABLE [INSTRUMENT CLUSTER].)

  • If there is not a DTC, replace the instrument cluster.

    (See INSTRUMENT CLUSTER REMOVAL/INSTALLATION.)
