SYMPTOM TROUBLESHOOTING [CAR-NAVIGATION SYSTEM]



Symptom

Description

CAR-NAVIGATION SYSTEM DOES NOT START OPERATION [CAR-NAVIGATION SYSTEM]

  • Even if the NAVI mode is selected, the car navigation system screen does not display

CAR-NAVIGATION SYSTEM DOES NOT OPERATION [CAR-NAVIGATION SYSTEM]

  • Even if the map screen is touched, the car navigation system cannot be operated

VEHICLE POSITION DEVIATES FROM THE ROUTE MAP [CAR-NAVIGATION SYSTEM]

  • The vehicle position on the map screen deviates from the actual vehicle position.

NO VOICE NAVIGATION [CAR-NAVIGATION SYSTEM]

  • Even if the route guidance mode is initiated, there is no voice guidance

DISPLAY SCREEN DOES NOT CHANGE TO THE NIGHT MODE [CAR-NAVIGATION SYSTEM]

  • Does not switch to the night screen even though the light switch is in the TNS position.
