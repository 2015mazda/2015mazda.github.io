POWER OUTER MIRROR SWITCH REMOVAL/INSTALLATION

NOTE:

  • The power outer mirror switch is integrated with the power window main switch.

1. Disconnect the negative battery cable. (See NEGATIVE BATTERY CABLE DISCONNECTION/CONNECTION [SKYACTIV-G 2.5].)
2. Remove the power window main switch. (See POWER WINDOW MAIN SWITCH REMOVAL/INSTALLATION.)
3. Install in the reverse order of removal.
