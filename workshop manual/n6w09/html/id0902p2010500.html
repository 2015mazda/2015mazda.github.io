DTC U0338:09 [FRONT BODY CONTROL MODULE (FBCM)]



Description

Error light switch signal received from start stop unit

Detection condition

  • Front body control module (FBCM) receives the CAN error signals from the start stop unit 3 times with the ignition switched ON (engine off or on).

Fail-safe function

  • If DTC U0338:09 is detected when the ignition is switched ON (engine off or on), the headlight LO and the TNS are illuminated.

  • If DTC U0338:09 is cleared, the headlight LO and the TNS are turned off, and the operation returns to normal.

Possible cause

  • Light switch malfunction

  • Start stop unit malfunction

  • Front body control module (FBCM) malfunction

System wiring diagram

Not applicable
