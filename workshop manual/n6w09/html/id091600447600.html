ROOF MOLDING INSTALLATION

1. Install the roof molding to the front T-studs.

2. Hook the roof molding to clip A and attach the roof molding while pressing it in the direction of the arrow shown in the figure.
3. Install the roof molding to the rear T-stud.
4. Press in the clips B installation positions shown in the figure and firmly adhere them to the body.
