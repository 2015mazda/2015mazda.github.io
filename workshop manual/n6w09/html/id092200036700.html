MAINTENANCE MONITOR OPERATION PROCEDURE

Except Canada Specs.

NOTE:

  • When any of the display conditions is met from the preset maintenance period, the maintenance monitor displays a message.

      • Display condition

    • Day-setting: Remaining days are 15 days or less

    • Distance setting: Remaining distance is 500 km/250 mile

  • The instrument cluster displays a message for which the display conditions are met 0.8 after the ignition is switched ON (engine off or on).

  • If several display conditions are met, the instrument cluster displays a message at approx. 5 s intervals according to the following order of precedence.

    • Scheduled Maintenance

    • Tire Rotation

    • Tire Pressure

  • The instrument cluster stores the maintenance monitor setting content, and the elapsed time and traveled distance from when the setting was made. The stored setting information is not cleared even if the battery is removed.

  • Because the maintenance monitor setting information stored in the instrument cluster cannot be written to a new instrument cluster, if the instrument cluster is replaced, the maintenance monitor is the initial set value. Before replacing the instrument cluster, explain to the customer that the maintenance monitor is reset to the initial setting.

Scheduled maintenance setting procedure
1. Switch the ignition ON (engine off or on).
2. Select "Setting" by operating the up/down switch and INFO button for the steering switch.
3. Select "Maintenance" by operating the up/down switch and INFO button for the steering switch.
4. Select "Scheduled" by operating the up/down switch and INFO button for the steering switch.
5. Select the following items and change the setting.
