ADAPTIVE FRONT LIGHTING SYSTEM (AFS) OFF SWITCH INSPECTION

NOTE:

  • The AFS OFF switch is integrated with the cluster switch.

  • For the AFS OFF switch removal/installation, refer to CLUSTER SWITCH REMOVAL/INSTALLATION. (See CLUSTER SWITCH REMOVAL/INSTALLATION.)

1. Remove the cluster switch. (See CLUSTER SWITCH REMOVAL/INSTALLATION.)
2. Verify that the resistance between cluster switch terminals B and C is as indicated in the table.

  • If the resistance can be verified as indicated in the table, go to the next step.

  • If not as indicated in the table, replace the cluster switch. (See CLUSTER SWITCH REMOVAL/INSTALLATION.)

3. Apply battery positive voltage to cluster switch terminal J, and connect terminal I to ground.

4. Verify that the LED illuminates.

  • If the LED does not illuminate, replace the cluster switch. (See CLUSTER SWITCH REMOVAL/INSTALLATION.)
