TRUNK BOARD REMOVAL/INSTALLATION

With Temporary Spare Tire

1. Set the trunk covering aside.
2. Remove trunk board No.1 in the direction of the arrow (1) shown in the figure.

3. Remove trunk board No.2 in the direction of the arrow (2) shown in the figure.
4. Remove trunk board No.3 in the direction of the arrow (3) shown in the figure.
5. Install in the reverse order of removal.

Without Temporary Spare Tire

1. Set the trunk covering aside.
2. Remove the trunk board in the direction of the arrow (1) shown in the figure.

3. Install in the reverse order of removal.
