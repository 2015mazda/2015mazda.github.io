2014 - Mazda6 - Body and Accessories

AUTO LEVEL CONTROL (ALC) FUNCTION IS INOPERATIVE [ENTIRE AUDIO SYSTEM]



Possible DTC

ALC function is inoperative

Using the M-MDS

U0155:00, U0010:88

Without using M-MDS (On-board diagnostic test mode)

16:Er12, 17:Er11

Possible cause

  • Open or short circuit in vehicle speed signal wiring harness (e. g., instrument cluster)

  • CAN signal wiring harness malfunction

  • Audio unit malfunction

NOTE:

  • Inspect the ALC function while driving the vehicle with playing the CD.

Diagnostic procedure

STEP

INSPECTION

ACTION

1

  • Turn the ALC function to ON.

  • Inspect the ALC function operation when driving the vehicle at ALC level 3.

  • Does the ALC system operate properly?

Yes

The system is normal. Explains the ALC function to the customer.

No

Go to the next step.

2

  • Retrieve the DTCs from all vehicle modules using the M-MDS.

  • Is there the following DTC displayed?

    • Vehicle speed signal wiring harness

    • CAN signal wiring harness

Yes

Go to the appropriate DTC inspection.

No

Replace the audio unit.

(See AUDIO UNIT REMOVAL/INSTALLATION.)
