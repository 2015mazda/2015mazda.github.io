SUPPLIER IDENTIFICATION PROCEDURE [AUDIO]

NOTE:

  • The unit supplier can vary. When asking the supplier (service center) for repair or replacement, identify the supplier and fill in the Audio Repair Order Form using the following procedures.

Verification procedure using label or inscribed lettering

1. Disconnect the negative battery cable. (See NEGATIVE BATTERY CABLE DISCONNECTION/CONNECTION [SKYACTIV-G 2.5].)
2. Remove the decoration panel. (See DECORATION PANEL REMOVAL/INSTALLATION.)
3. Remove the audio unit. (See AUDIO UNIT REMOVAL/INSTALLATION.)
4. Verify the supplier name by referring to the label attached to the unit.

Audio unit (with color LCD)

Audio unit (without color LCD)

Verification procedure using on-board diagnostic assist function

Audio unit (with color LCD)
1. Switch the ignition to ACC or ON (engine off).
2. Verify that the audio is turned on. If the audio is turned off, press the POWER/VOLUME button to turn it on.
3. Press the POWER/VOLUME button and SEEK button (up) simultaneously to launch the diagnostic assist function.

4. Verify that the launched diagnostic assist function is displayed on the screen.

5. Input diagnostic assist code "06".

6. Select "ENTER button".
7. Identify the supplier code by referring to the display.

Audio unit (without color LCD)
1. Switch the ignition to ACC or ON (engine off).
2. Verify that the audio is turned on. If the audio is turned off, press the POWER/VOLUME button to turn it on.
3. Press PRESET button 5 for 3 s or more while simultaneously pressing the POWER/VOLUME button to launch the supplier verification mode.

4. Identify the supplier code by referring to the audio display.
