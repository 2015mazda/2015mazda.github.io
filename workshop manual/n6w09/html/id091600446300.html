2014 - Mazda6 - Body and Accessories

FRONT DOOR GARNISH REMOVAL/INSTALLATION

WARNING:

  • Using a utility knife with bare hands can cause injury. Always wear gloves when using a utility knife.

1. Remove the front beltline molding. (See FRONT BELTLINE MOLDING REMOVAL/INSTALLATION.)
2. Using a tape-wrapped fastener remover, remove clips A in the direction of arrow (1), (2) shown in the figure.

3. Remove the double-sided adhesive tape in the position shown in the figure using the utility knife, remove the front door garnish from the body.
4. Install in the reverse order of removal. (See Front Door Garnish Installation Note.)

Front Door Garnish Installation Note

1. When installing a new front door garnish, perform the following procedure:
a. Remove the double-sided adhesive tape on the front door sash.
b. Remove any grease or dirt from the affixing surface of the front door sash.
