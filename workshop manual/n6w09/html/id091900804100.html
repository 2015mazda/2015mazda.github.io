WIPER/WASHER SYSTEM PERSONALIZATION FEATURES SETTING PROCEDURE

Using M-MDS

  • The auto wiper function can be enabled or disabled using the M-MDS. Refer to the control system personalization features setting. (See CONTROL SYSTEM PERSONALIZATION FEATURES SETTING PROCEDURE.)

Using audio unit (with color LCD)

  • The user can enable or disable the auto wiper function by operating the audio unit display. Refer to the entertainment system personalization features setting. (See AUDIO UNIT PERSONALIZATION FEATURES SETTING PROCEDURE.)
