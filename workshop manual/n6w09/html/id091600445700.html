REAR DOOR GARNISH REMOVAL/INSTALLATION

WARNING:

  • Using a utility knife with bare hands can cause injury. Always wear gloves when using a utility knife.

Front

1. Remove the rear beltline molding. (See REAR BELTLINE MOLDING REMOVAL/INSTALLATION.)
2. Using a tape-wrapped fastener remover, remove clips A in the direction of arrow (1), (2) shown in the figure.

3. Remove the double-sided adhesive tape in the position shown in the figure using the utility knife, remove the rear door garnish from the rear door sash.
4. Install in the reverse order of removal. (See Rear Door Garnish Installation Note.)

Rear

1. Remove the rear beltline molding. (See REAR BELTLINE MOLDING REMOVAL/INSTALLATION.)
2. Using a tape-wrapped fastener remover, remove clips B in the direction of arrow (1), (2) shown in the figure.

3. Remove the double-sided adhesive tape in the position shown in the figure using the utility knife, remove the rear door garnish from the rear door sash.
4. Install in the reverse order of removal. (See Rear Door Garnish Installation Note.)

Rear Door Garnish Installation Note

1. When installing a new rear door garnish, perform the following procedure:
a. Remove the double-sided adhesive tape on the rear door sash.
b. Remove any grease or dirt from the affixing surface of the rear door sash.
