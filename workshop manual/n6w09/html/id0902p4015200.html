2014 - Mazda6 - Body and Accessories

DTC U0010:88/U0155:00 [REAR BODY CONTROL MODULE (RBCM)]



Description

  • U0010:88

    • Unit communication error (MS-CAN)

  • U0155:00

    • Communication error with instrument cluster

Detection condition

  • U0010:88

    • Rear body control module (RBCM) detects a CAN communication line (MS-CAN) error with the ignition switched ON (engine off or on).

  • U0155:00

    • Rear body control module (RBCM) cannot receive the CAN signals from the instrument cluster for 5 s or more with the ignition switched ON (engine off or on).

Fail-safe function

Not applicable

Possible cause

  • Malfunction in the CAN bus communication line

  • Malfunction in the CAN line between instrument cluster and rear body control module (RBCM)

System wiring diagram

Not applicable

Diagnostic Procedure

  • Perform the malfunction diagnosis according to the troubleshooting procedure for the multiplex communication system. (See FOREWORD.)
