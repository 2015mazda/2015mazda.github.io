SUNROOF SWITCH REMOVAL/INSTALLATION

NOTE:

  • The sunroof switch is together with the front map light.

1. Disconnect the negative battery cable. (See NEGATIVE BATTERY CABLE DISCONNECTION/CONNECTION [SKYACTIV-G 2.5].)
2. Remove the front map light. (See MAP LIGHT REMOVAL/INSTALLATION.)
3. Install in the reverse order of removal.
