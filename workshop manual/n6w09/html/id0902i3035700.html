2014 - Mazda6 - Body and Accessories

DTC U0001:88/U0100:00/U0131:00/U0155:00/U0214:00 [BLIND SPOT MONITORING (BSM)]



Description

  • U0001:88

    • Module communication error (HS-CAN)

  • U0100:00

    • Communication error with PCM

  • U0131:00

    • Communication error with EPS control module

  • U0155:00

    • Communication error with instrument cluster

  • U0214:00

    • Communication error with start stop unit

Detection condition

  • U0001:88

    • BSM control module detected a CAN bus communication line (HS-CAN) malfunction for 0.4 s or more.

  • U0100:00

    • BSM control module cannot receive a CAN signal from the PCM for 1.5 s or more.

  • U0131:00

    • BSM control module cannot receive a CAN signal from the EPS control module for 1.5 s or more.

  • U0155:00

    • BSM control module cannot receive a CAN signal from the instrument cluster.

  • U0214:00

    • BSM control module cannot receive a CAN signal from the start stop unit for 1.5 s or more.

Fail-safe function

  • U0001:88

    • Blind spot monitoring (BSM) system is stopped.

  • U0100:00

    • Blind spot monitoring (BSM) system is stopped.

  • U0131:00

    • Blind spot monitoring (BSM) system is stopped.

  • U0155:00

    • Blind spot monitoring (BSM) system is stopped.

  • U0214:00

    • Blind spot monitoring (BSM) system is stopped.

Possible cause

  • Malfunction in CAN bus communication line

  • Malfunction in CAN communication line between PCM and BSM control module

  • Malfunction in CAN communication line between EPS control module and BSM control module

  • Malfunction in CAN communication line between instrument cluster and BSM control module

  • Malfunction in CAN communication line between start stop unit and BSM control module

System wiring diagram

Not applicable

Diagnostic Procedure

  • Perform the malfunction diagnosis according to the troubleshooting procedure for the multiplex communication system. (See FOREWORD.)
