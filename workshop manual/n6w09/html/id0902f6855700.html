DTC TABLE [AUDIO]

NOTE:

  • There are two types of procedures for the audio unit (with color LCD) inspection; one using the M-MDS and the other which does not (on-board diagnostic test mode). Because the DTCs differ if using or not using the M-MDS, perform the DTC inspection using the M-MDS method and the method not using it. (See DTC INSPECTION [AUDIO].)
