2014 - Mazda6 - HVAC

REFRIGERANT PRESSURE SENSOR INSPECTION [FULL-AUTO AIR CONDITIONER]

1. Install the manifold gauge.
2. Verify the high-pressure side reading of the manifold gauge.
3. Measure the terminal voltage at PCM terminal 2BB, 2AX and 2AJ.
4. Using the graph below, measure and verify the terminal voltages at 2AX.
5. Follow the PCM inspection when measuring the other terminal voltages. (See PCM INSPECTION [SKYACTIV-G 2.5].)

  • Terminal 2BB: Approx. 5 V (Ignition switch ON)

  • Terminal 2AJ: 1.0 V or less

    • If each voltage is not normal, inspect the related wiring harness.

      • If there is any malfunction, replace the related wiring harness.

      • If the wiring harness is normal, replace the refrigerant pressure sensor.
