2014 - Mazda6 - Restraints

DTC U0001:88/U0155:00 [TWO-STEP DEPLOYMENT CONTROL SYSTEM]



System malfunction location

  • U0001:88: Module communication error (HS-CAN)

  • U0155:00: Communication error with instrument cluster

Detection condition

    U0001:88

  • The SAS control module detects a CAN communication line (HS-CAN) error.

    U0155:00

  • The SAS control module could not receive CAN signal from the instrument cluster for 5 s or more.

Fail-safe

Not applicable

Possible cause

  • Malfunction in CAN bus communication line

  • Malfunction in CAN line between the SAS control module and instrument cluster

System wiring diagram

Not applicable

Diagnostic Procedure

  • Perform the malfunction diagnosis according to the troubleshooting procedure for the multiplex communication system. (See FOREWORD.)
