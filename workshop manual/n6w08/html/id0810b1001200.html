2014 - Mazda6 - Restraints

SEAT WEIGHT SENSOR REMOVAL/INSTALLATION [TWO-STEP DEPLOYMENT CONTROL SYSTEM]

CAUTION:

  • When the seat weight sensor is replaced with a new one, perform the seat weight sensor calibration using the M–MDS. (See AIR BAG SYSTEM SERVICE WARNINGS [TWO-STEP DEPLOYMENT CONTROL SYSTEM].) (See AIR BAG SYSTEM SERVICE CAUTIONS [TWO-STEP DEPLOYMENT CONTROL SYSTEM].)

  • When replacing the seat weight sensor, replace it together with the passenger-side seat slide adjuster (because seat weight sensor is integrated with passenger-side seat slide adjuster).

    (See FRONT SEAT REMOVAL/INSTALLATION.)
