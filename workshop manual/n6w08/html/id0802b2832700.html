2014 - Mazda6 - Restraints

DTC B00A0:4A [TWO-STEP DEPLOYMENT CONTROL SYSTEM]



System malfunction location

  • Seat weight sensor ID mismatch

Detection condition

WARNING:

  • Detection conditions are for understanding the DTC outline before performing an inspection. Performing an inspection according to only the detection conditions may cause injury due to an operating error, or damage the system. When performing an inspection, always follow the inspection procedure.

  • Seat weight sensor ID mismatch

Fail-safe

Not applicable

Possible cause

  • Seat weight sensor control module malfunction

  • SAS control module malfunction

System wiring diagram

Not applicable

Diagnostic Procedure



Step

Inspection

Action

1

 PERFORM SEAT WEIGHT SENSOR CONTROL MODULE DTC INSPECTION

  • Clear the DTC for the SAS control module using the M-MDS. (See CLEARING DTC [TWO-STEP DEPLOYMENT CONTROL SYSTEM].)

  • Perform the DTC inspection for the SAS control module using the M-MDS. (See DTC INSPECTION [TWO-STEP DEPLOYMENT CONTROL SYSTEM].)

  • Is the same DTC present?

Yes

Replace the seat weight sensor control module. (See SEAT WEIGHT SENSOR CONTROL MODULE REMOVAL/INSTALLATION [TWO-STEP DEPLOYMENT CONTROL SYSTEM].)

No

Go to the next step

2

 PERFORM SAS CONTROL MODULE DTC INSPECTION

  • Clear the DTC for the SAS control module using the M-MDS. (See CLEARING DTC [TWO-STEP DEPLOYMENT CONTROL SYSTEM].)

  • Perform the DTC inspection for the SAS control module using the M-MDS. (See DTC INSPECTION [TWO-STEP DEPLOYMENT CONTROL SYSTEM].)

  • Is the same DTC present?

  • Are there other DTCs output?

Yes

Replace the SAS control module. (See SAS CONTROL MODULE REMOVAL/INSTALLATION [TWO-STEP DEPLOYMENT CONTROL SYSTEM].)

No

DTC troubleshooting completed.
