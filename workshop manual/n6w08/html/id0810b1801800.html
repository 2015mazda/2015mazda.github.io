2014 - Mazda6 - Restraints

SEAT WEIGHT SENSOR INSPECTION [TWO-STEP DEPLOYMENT CONTROL SYSTEM]

CAUTION:

  • If any of the following work is performed, perform the seat weight sensor inspection using the M-MDS.

    • Removal of the passenger-side seat

    • Loosening and retightening of passenger’s seat fixing bolts

    • Or, the vehicle was involved in a collision

  • If any of the following work is performed, perform the seat weight sensor calibration using the M-MDS. (See SEAT WEIGHT SENSOR CALIBRATION [TWO-STEP DEPLOYMENT CONTROL SYSTEM].)

    • Replacement with a new seat weight sensor

    • Replacement with a new seat weight sensor control module

    • Replacement with new passenger-side seat parts

    • Disassembly of the passenger-side seat

1. Connect the M-MDS to the DLC-2.
2. Verify that SAS control module DTC B00A0:28 is detected using the M-MDS.

  • If DTC B00A0:28 is detected, perform the on-board diagnostic system procedure. (See DTC B00A0:28 [TWO-STEP DEPLOYMENT CONTROL SYSTEM].)
