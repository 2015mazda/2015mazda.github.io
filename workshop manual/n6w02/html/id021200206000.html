2014 - Mazda6 - Suspension

TIRE PRESSURE MONITORING SYSTEM SET SWITCH INSPECTION

1. Remove the TPMS set switch. (See TIRE PRESSURE MONITORING SYSTEM SET SWITCH REMOVAL/INSTALLATION.)
2. Verify that the resistance between the TPMS set switch terminals B and C is as indicated in the table.

  • If not as indicated in the table, replace the TPMS set switch.

TPMS set switch specification

Terminal

Test condition

Resistance (ohms)

B—C

TPMS set switch pressed

187.61—191.39

All switches released

1139.99—1163.01

3. Apply battery voltage to TPMS set switch terminal J, and connect terminal I to ground.
4. Verify that the LED illuminates.

  • If there is any malfunction, replace the TPMS set switch.
