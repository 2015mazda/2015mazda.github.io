DTC P2228:00 [SKYACTIV-G 2.5]



DTC P2228:00

BARO sensor circuit low input

DETECTION CONDITION

  • The PCM monitors input voltage from the BARO sensor. If the input voltage is below the specified value for 5 s, the PCM determines that the BARO sensor circuit has a malfunction.

    Diagnostic support note

  • This is a continuous monitor (CCM).

  • The check engine light illuminates if the PCM detects the above malfunction condition during the first drive cycle.

  • FREEZE FRAME DATA (Mode 2)/Snapshot data is available.

  • DTC is stored in the PCM memory.

FAIL-SAFE FUNCTION

  • Fixes the barometric pressure for the engine control at 101.32 kPa {1.0332 kgf/cm2, 14.695 psi}.

POSSIBLE CAUSE

  • BARO sensor (built-into PCM) malfunction

  • PCM malfunction

SYSTEM WIRING DIAGRAM

Not applicable

CAUTION:

  • Verify the malfunction symptom according to not only the PID value but also the symptom troubleshooting.
