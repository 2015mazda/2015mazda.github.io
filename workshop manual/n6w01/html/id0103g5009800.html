2014 - Mazda6 - Engine

TROUBLESHOOTING INDEX [VEHICLE CONTROL MODULE (V/C-MODULE)]



Troubleshooting item

Description

MRCC WARNING LIGHT (AMBER) IS NOT ILLUMINATED [VEHICLE CONTROL MODULE (V/C-MODULE)]

  • The MRCC warning light (amber) is not illuminated when the ignition is switched ON (engine off or on).

MRCC WARNING LIGHT (AMBER) REMAINS ILLUMINATED [VEHICLE CONTROL MODULE (V/C-MODULE)]

  • The MRCC warning light (amber) remains illuminated after switching the ignition ON (engine off or on).

MRCC INDICATOR LIGHT (GREEN) IS NOT ILLUMINATED DURING MRCC OPERATION [VEHICLE CONTROL MODULE (V/C-MODULE)]

  • The MRCC indicator light (green) is not illuminated even if the MRCC is operated.

SYSTEM SCREEN IS NOT DISPLAYED IN LCD / LCD DISPLAY IS UNMATCHED [VEHICLE CONTROL MODULE (V/C-MODULE)]

  • The system screen is displayed in the LCD even if the system is operated.

  • The display in the LCD is unmatched with the system operation.

MRCC/FOW MIS-OPERATION [VEHICLE CONTROL MODULE (V/C-MODULE)]

  • The system operates on a vehicle traveling in the adjacent drive lane.

  • System operates on objects not within the millimeter wave emission range of the radar sensors.
