2014 - Mazda6 - Engine

FUEL PUMP CONTROL MODULE REMOVAL/INSTALLATION [SKYACTIV-G 2.5]

1. Disconnect the negative battery cable. (See NEGATIVE BATTERY CABLE DISCONNECTION/CONNECTION [SKYACTIV-G 2.5].)
2. Tip the rear seat back (LH). (See REAR SEAT BACK REMOVAL/INSTALLATION.)
3. Remove in the order indicated in the table.



1

Fuel pump control module connector

2

Fuel pump control module

3

Fuel pump control module bracket

4. Install in the reverse order of removal.
