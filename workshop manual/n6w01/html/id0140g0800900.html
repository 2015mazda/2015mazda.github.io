2014 - Mazda6 - Engine

MANIFOLD ABSOLUTE PRESSURE (MAP) SENSOR INSPECTION [SKYACTIV-G 2.5]

Voltage Inspection

1. Connect the M-MDS to the DLC–2.
2. Switch the ignition ON (engine off).
3. Verify that the MAP sensor output voltage (PID: MAP) is within the specification. (See ON-BOARD DIAGNOSTIC TEST [SKYACTIV-G 2.5].) (See PCM INSPECTION [SKYACTIV-G 2.5].)

  • If not within the specification, replace the MAP sensor/IAT sensor No.2. (See MANIFOLD ABSOLUTE PRESSURE (MAP) SENSOR/INTAKE AIR TEMPERATURE (IAT) SENSOR NO.2 REMOVAL/INSTALLATION [SKYACTIV-G 2.5].)
