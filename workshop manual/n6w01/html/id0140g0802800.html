KNOCK SENSOR (KS) INSPECTION [SKYACTIV-G 2.5]

Resistance Inspection

1. Disconnect the negative battery cable. (See NEGATIVE BATTERY CABLE DISCONNECTION/CONNECTION [SKYACTIV-G 2.5].)
2. Remove the plug hole plate. (See PLUG HOLE PLATE REMOVAL/INSTALLATION [SKYACTIV-G 2.5].)
3. Remove the intake manifold. (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION [SKYACTIV-G 2.5].)
4. Disconnect the KS connector.
5. Remove the KS. (See KNOCK SENSOR (KS) REMOVAL/INSTALLATION [SKYACTIV-G 2.5].)
6. Measure the resistance between KS terminals A and B.

  • If not as specified, replace the KS. (See KNOCK SENSOR (KS) REMOVAL/INSTALLATION [SKYACTIV-G 2.5].)

    Specification

  • 504—616 kilohms [10—30 °C {50—86 °F}]
