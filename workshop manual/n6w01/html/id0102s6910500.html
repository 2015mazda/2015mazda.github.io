DTC U0298:00 [SKYACTIV-G 2.5]

Details On DTCs



DTC U0298:00

CAN communication: communication error to DC-DC converter

DETECTION CONDITION

  • PCM detects a communication error between the PCM and DC-DC converter.

    Diagnostic support note

  • This is a continuous monitor (other).

  • The check engine light does not illuminate.

  • FREEZE FRAME DATA (Mode 2)/Snapshot data is not available.

  • The DTC is stored in the PCM memory.

FAIL-SAFE FUNCTION

  • Not applicable

VEHICLE STATUS WHEN DTCs ARE OUTPUT

  • Illuminates the charging system warning light

POSSIBLE CAUSE

  • Communication line between DC-DC converter and PCM malfunction

  • DC-DC converter malfunction

  • PCM malfunction

SYSTEM WIRING DIAGRAM

Not applicable

Repeatability Verification Procedure

  • Clear the DTC from the PCM memory using the M-MDS. (See CLEARING DTC [SKYACTIV-G 2.5].)

  • Start the engine.
