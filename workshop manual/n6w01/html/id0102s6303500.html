2014 - Mazda6 - Engine

DTC P2110:00 [SKYACTIV-G 2.5]



DTC P2110:00

Throttle valve actuator control system-forced limited RPM

DETECTION CONDITION

  • PCM internal malfunction.

    Diagnostic support note

  • This is a continuous monitor (CCM).

  • The check engine light illuminates if the PCM detects the above malfunction condition during the first drive cycle.

  • FREEZE FRAME DATA (Mode 2)/Snapshot data is available.

  • DTC is stored in the PCM memory.

FAIL-SAFE FUNCTION

  • Restricts the upper limit of the engine speed.

  • Stops the drive-by-wire control (throttle valve is open at approx. 8 ° by return spring force).

POSSIBLE CAUSE

  • Throttle body malfunction

  • PCM malfunction

    • Throttle valve actuator control module internal processor error (built-into PCM)

    • Fuel injector control driver internal processor error (built-into PCM)

SYSTEM WIRING DIAGRAM

Not applicable
