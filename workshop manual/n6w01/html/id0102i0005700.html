2014 - Mazda6 - Engine

DTC U0001:88, U0100:00, U0101:00, U0121:00, U0131:00, U0140:00, U0151:00, U0155:00, U0214:00, U0235:68, U023A:00 [VEHICLE CONTROL MODULE (V/C-MODULE)]



System malfunction location

  • U0001:88: Module communication error (HS-CAN)

  • U0100:00: Communication error with PCM

  • U0101:00: Communication error with TCM

  • U0121:00: Communication error with DSC HU/CM

  • U0131:00: Communication error with EPS control module

  • U0140:00: Communication error with front body control module (FBCM)

  • U0151:00: Communication error with SAS control module

  • U0155:00: Communication error with Instrument cluster

  • U0214:00: Communication error with start stop unit

  • U0235:68: Communication error with laser sensor

  • U023A:00: Communication error with forward sensing camera (FSC)

Detection condition

  • U0001:88: Vehicle control module (V/C-module) detects a CAN bus communication line (HS-CAN) malfunction.

  • U0100:00: Vehicle control module (V/C-module) cannot receive the CAN signal from the PCM.

  • U0101:00: Vehicle control module (V/C-module) cannot receive the CAN signal from the TCM.

  • U0121:00: Vehicle control module (V/C-module) cannot receive the CAN signal from the DSC HU/CM.

  • U0131:00: Vehicle control module (V/C-module) cannot receive the CAN signal from the EPS control module.

  • U0140:00: Vehicle control module (V/C-module) cannot receive the CAN signal from the front body control module (FBCM).

  • U0151:00: Vehicle control module (V/C-module) cannot receive the CAN signal from the SAS control module.

  • U0155:00: Vehicle control module (V/C-module) cannot receive the CAN signal from the Instrument cluster.

  • U0214:00: Vehicle control module (V/C-module) cannot receive the CAN signal from the start stop unit.

  • U0235:68: Vehicle control module (V/C-module) cannot receive the CAN signal from the laser sensor.

  • U023A:00: Vehicle control module (V/C-module) cannot receive the CAN signal from the forward sensing camera (FSC).

  • Malfunction in CAN line between PCM and vehicle control module (V/C-module)
