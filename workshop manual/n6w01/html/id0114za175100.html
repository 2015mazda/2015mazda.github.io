2014 - Mazda6 - Engine

FUEL PUMP CONTROL MODULE INSPECTION [SKYACTIV-G 2.5]

1. Perform the KOEO self-test. (See KOEO/KOER SELF TEST [SKYACTIV-G 2.5].)
2. Verify that a fuel pump control module related DTC is detected. (See ON-BOARD DIAGNOSTIC TEST [SKYACTIV-G 2.5].)

  • If DTCs are detected, repair the malfunctioning part according to the applicable DTC troubleshooting. (See DTC TABLE [SKYACTIV-G 2.5].)

  • If a DTC is not detected, go to the next step.

3. Inspect the following:

  • Fuel pump control module damage

  • Fuel pump control module connector damage or corrosion

    • If there is any malfunction, repair or replace the malfunctioning part.
