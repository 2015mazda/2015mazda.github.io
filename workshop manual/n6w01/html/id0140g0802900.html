KNOCK SENSOR (KS) REMOVAL/INSTALLATION [SKYACTIV-G 2.5]

1. Disconnect the negative battery cable. (See NEGATIVE BATTERY CABLE DISCONNECTION/CONNECTION [SKYACTIV-G 2.5].)
2. Remove the plug hole plate. (See PLUG HOLE PLATE REMOVAL/INSTALLATION [SKYACTIV-G 2.5].)
3. Remove the intake manifold. (See INTAKE-AIR SYSTEM REMOVAL/INSTALLATION [SKYACTIV-G 2.5].)
4. Disconnect the KS connector.
5. Remove the KS.

6. Install in the reverse order of removal. (See KS Installation Note.)

KS Installation Note

  • Verify that there is no foreign matter caught between the KS unit and cylinder block.

  • The direction of the KS connector is within the 60-degree angle range shown in the figure.
