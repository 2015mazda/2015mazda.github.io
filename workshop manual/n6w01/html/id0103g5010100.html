2014 - Mazda6 - Engine

MRCC WARNING LIGHT (AMBER) REMAINS ILLUMINATED [VEHICLE CONTROL MODULE (V/C-MODULE)]

Description

  • The MRCC warning light (amber) remains illuminated after switching the ignition ON (engine off or on).

Possible malfunction

  • Instrument cluster cannot turn off the MRCC warning light (amber).

    • Instrument cluster malfunction (malfunction in MRCC warning light (amber) illumination circuit)

  • The vehicle control module (V/C-module) cannot turn off the MRCC warning light (amber).

    • vehicle control module (V/C-module) detects system malfunction

    • vehicle control module (V/C-module) malfunction (false sending of MRCC warning light (amber) illumination request)

Diagnostic Procedure



Step

Inspection

Action

1
    INSPECT INSTRUMENT CLUSTER

  • Turn off the MRCC warning light (amber) using the M-MDS instrument cluster active command modes WL+IL.

    (See ACTIVE COMMAND MODES INSPECTION [INSTRUMENT CLUSTER].)

  • Are all the warning lights turned on/off?

Yes

Go to the next step.

No

Replace the instrument cluster.

(See INSTRUMENT CLUSTER REMOVAL/INSTALLATION.)

2
    INSPECT VEHICLE CONTROL MODULE (V/C-MODULE)

  • Inspect the vehicle control module (V/C-module).

    (See VEHICLE CONTROL MODULE (V/C-MODULE) INSPECTION [MAZDA RADAR CRUISE CONTROL (MRCC) SYSTEM].)

  • Is the vehicle control module (V/C-module) normal?

Yes

Go to the next step.

No

Replace the vehicle control module (V/C-module).

(See VEHICLE CONTROL MODULE (V/C-MODULE) REMOVAL/INSTALLATION [MAZDA RADAR CRUISE CONTROL (MRCC) SYSTEM].)

3
    VERIFY MRCC WARNING LIGHT (AMBER) ILLUMINATION CONDITION

  • Switch the ignition ON (engine off or on).

  • After switching the ignition ON (engine off or on), does the MRCC warning light (amber) remain illuminated even after 5 s or more have elapsed?

    NOTE:

    • After switching the ignition ON (engine off or on), the MRCC warning light (amber) is illuminated for approx. 3 s for MRCC warning light (amber) illumination verification.

Yes

Replace the vehicle control module (V/C-module).

(See VEHICLE CONTROL MODULE (V/C-MODULE) REMOVAL/INSTALLATION [MAZDA RADAR CRUISE CONTROL (MRCC) SYSTEM].)

No

Symptom troubleshooting is completed.
