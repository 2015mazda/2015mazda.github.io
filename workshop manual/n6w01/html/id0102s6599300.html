DTC P130C:00 [SKYACTIV-G 2.5]



DTC P130C:00

Pre-ignition detected

DETECTION CONDITION

  • Pre-ignition is detected a few times continuously with the engine coolant temperature at 80 °C {176 °F} or more.

    Diagnostic support note

  • This is a continuous monitor (other).

  • The check engine light illuminates if the PCM detects the above malfunction condition during the first drive cycle.

  • FREEZE FRAME DATA (Mode 2)/Snapshot data is available.

  • DTC is stored in the PCM memory.

FAIL-SAFE FUNCTION

Not applicable

POSSIBLE CAUSE

  • Fuel having octane rating lower than specified fuel is used

  • Carbon accumulated in combustion chamber

  • PCM malfunction

SYSTEM WIRING DIAGRAM

Not applicable

CAUTION:

  • Verify the malfunction symptom according to not only the PID value but also the symptom troubleshooting.
