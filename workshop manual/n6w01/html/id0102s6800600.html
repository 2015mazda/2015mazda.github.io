2014 - Mazda6 - Engine

OBD-II ON-BOARD SYSTEM READINESS TEST [SKYACTIV-G 2.5]

  • This shows the OBD-II systems operating status. If any monitor function is incomplete, the M-MDS will identify which monitor function has not been completed. Misfires, Fuel System and Comprehensive Components (CCM) are continuous monitoring-type functions. The catalyst, evaporation system and oxygen sensor will be monitored under drive cycles. The OBD-II diagnostic system is initialized by performing the DTC cancellation procedure or disconnecting the negative battery cable.
