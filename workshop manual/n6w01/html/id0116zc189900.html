2014 - Mazda6 - Engine

CHECK VALVE INSPECTION [SKYACTIV-G 2.5]

U.S.A. And CANADA

1. Remove the check valve. (See FUEL TANK REMOVAL/INSTALLATION [SKYACTIV-G 2.5].)
2. Verify that there is airflow when positive pressure is applied to port A.

  • If there is no airflow, replace the breather pipe. (See FUEL TANK REMOVAL/INSTALLATION [SKYACTIV-G 2.5].)

3. Verify that there is airflow when negative pressure is applied to port A.

  • If there is no airflow, replace the breather pipe. (See FUEL TANK REMOVAL/INSTALLATION [SKYACTIV-G 2.5].)

Except U.S.A. And CANADA

1. Remove the check valve. (See CHECK VALVE REMOVAL/INSTALLATION [SKYACTIV-G 2.5].)
2. Verify that there is no airflow when positive pressure is applied to port A.

  • If there is airflow, replace the check valve. (See CHECK VALVE REMOVAL/INSTALLATION [SKYACTIV-G 2.5].)

3. Verify that there is no airflow when negative pressure is applied to port A.

  • If there is no airflow, replace the check valve. (See CHECK VALVE REMOVAL/INSTALLATION [SKYACTIV-G 2.5].)
