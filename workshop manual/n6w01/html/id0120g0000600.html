VEHICLE CONTROL MODULE (V/C-MODULE) REMOVAL/INSTALLATION [MAZDA RADAR CRUISE CONTROL (MRCC) SYSTEM]

NOTE:

  • The vehicle control module (V/C-module) prior to replacement stores the vehicle specification information.

  • The new vehicle control module (V/C-module) does not store any vehicle specification information.

  • When the ignition is switched ON after the vehicle control module (V/C-module) is replaced, the vehicle control module (V/C-module) reads the vehicle specification information sent via CAN transmission from the instrument cluster, and stores the vehicle specification information.

  • The automatic configuration function is only performed when the vehicle control module (V/C-module) is newly replaced.

1. Disconnect the negative battery cable. (See NEGATIVE BATTERY CABLE DISCONNECTION/CONNECTION [SKYACTIV-G 2.5].)
2. Remove the following parts:
a. Front scuff plate (RH) (See FRONT SCUFF PLATE REMOVAL/INSTALLATION.)
b. Front side trim (RH) (See FRONT SIDE TRIM REMOVAL/INSTALLATION.)
3. Disconnect the vehicle control module (V/C-module) connector.

4. Remove the bolts.
5. Remove the vehicle control module (V/C-module).
6. Install in the reverse order of removal.
7. When newly replacing the vehicle control module (V/C-module), switch the ignition ON and perform the vehicle control module (V/C-module) automatic configuration.
8. Perform the radar sensor aiming adjustment. (See RADAR SENSOR AIMING [MAZDA RADAR CRUISE CONTROL (MRCC) SYSTEM].)
9. Clear the DTCs. (See CLEARING DTC [VEHICLE CONTROL MODULE (V/C-MODULE)].)
