Features



Improved safety

  • Intrusion minimizing brake pedal adopted

  • Antilock Brake System (ABS) adopted

  • Electronic Brakeforce Distribution (EBD) control adopted

  • Dynamic Stability Control (DSC) adopted

  • Electrical brake assist control adopted

  • Hill Launch Assist (HLA) adopted

  • Tire Pressure Monitoring System (TPMS) adopted

  • Smart City Brake Support (SCBS) adopted (vehicles with Smart City Brake Support (SCBS))

Improved braking force

  • Large diameter front disc brakes adopted

  • Large diameter rear disc brakes adopted

  • Vacuum pump adopted

Improved serviceability

  • Enhanced malfunction diagnosis system for use with Mazda Modular Diagnostic System (M-MDS)

  • Center lever type parking brake, adjustable from vehicle interior, adopted

Size and weight reduction

  • Integrated construction of the Hydraulic Unit (HU) and Control Module (CM) adopted for the DSC HU/CM

Improved reliability

  • Semi-conductor element type ABS wheel-speed sensor adopted

  • Magnetic encoder type ABS sensor rotor adopted

  • DSC HU/CM with built-in brake fluid pressure sensor

  • Receives DSC control signal from Sophisticated Air bag Sensor (SAS) control module instead of conventional combined sensor
