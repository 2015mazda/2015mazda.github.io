2014 - Mazda6 - Brakes

CONTROLLER AREA NETWORK (CAN)

Outline

  • The DSC HU/CM sends and receives data to and from other modules via the CAN system. Refer to MULTIPLEX COMMUNICATION SYSTEM for a detailed explanation of the CAN system. (See CONTROLLER AREA NETWORK (CAN) SYSTEM.)

Data sent

  • Cruise control system-related information

  • DSC system-related information

  • Smart City Brake Support (SCBS)-related information

  • Vehicle speed

  • Wheel speed (LF, RF, LR, RR)

  • Yaw rate

Data received

  • Accelerator pedal opening angle

  • Ambient temperature

  • Back-up light on request

  • Back-up light ON signal (ATX)

  • Back-up light ON signal (MTX)

  • Brake fluid level

  • Brake light status

  • Brake prefill request signal (SCBS)

  • Brake switch (No.2 signal)

  • Cluster switch status

  • Clutch pedal stroke sensor

  • Collision detection (front, side, roll over)

  • Collision detection (rear)

  • Cruise control control status

  • Cruise control set speed

  • Cruise control switch

  • Cruise control system-related information

  • DSC OFF switch status

  • Engine displacement

  • Engine exhaust volume

  • Engine speed

  • Engine torque

  • Fuel cut request

  • Manual shift control

  • Neutral switch

  • Parking brake status

  • Shift range position

  • Smart City Brake Support (SCBS)-related information

  • Steering angle (relative angle) signal

  • TPMS set switch status

  • Tire size

  • Vehicle speed
