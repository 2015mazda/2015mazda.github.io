CRUISE CONTROL SWITCH [MAZDA RADAR CRUISE CONTROL (MRCC) SYSTEM]

Purpose

  • The cruise control switch can switch the MRCC system on/off and perform various settings.

Function

  • The cruise control switch sends electric signals for each switch operation to the start stop unit.

Construction

  • Installed to right-hand spoke of the steering wheel.

Function of Buttons



No.

Name

Function

1

Cancel switch

Cancels the MRCC system

2

RESUME switch

Resets the recorded set vehicle speed and begins control.

3

Upper switch

  • Sets vehicle speed

  • Increases set vehicle speed

    • Increases speed in approx. 2 km/h {1 mph} increments each time switch is short pressed

    • Increases speed in approx. 8 km/h {5 mph} increments each time switch is long pressed

Lower switch

  • Sets vehicle speed

  • Decreases set vehicle speed

    • Decreases speed in approx. 2 km/h {1 mph} increments each time switch is short pressed

    • Decreases speed in approx. 8 km/h {5 mph} increments each time switch is long pressed

4

MRCC ON switch

Turns on the MRCC system

5

MRCC OFF switch

Turns off the MRCC system

6

DISTANCE switch

Sets the distance between vehicles for the headway control (4 stages: Long, medium, short, extremely short)

Fail-safe

  • Function not equipped.
