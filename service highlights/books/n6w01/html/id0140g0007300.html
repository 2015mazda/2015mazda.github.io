2014 - Mazda6 - Engine

ACCELERATOR PEDAL POSITION (APP) SENSOR [SKYACTIV-G 2.5]

Purpose/Function

  • Detects the accelerator pedal depression amount as basic information for mainly determining the throttle valve opening.

  • Detects the accelerator pedal depression amount and sends it to the PCM as an accelerator pedal opening angle signal.

Construction

  • Built into the accelerator pedal.

  • A Hall element has been adopted as the detecting element for detecting the accelerator pedal depression amount.

  • The APP sensor has a main sensor (No.1) and a sub sensor (No.2) for improved detection accuracy.

Operation

  • Sends the signal, which fluctuates according to the throttle valve opening/closing, to the PCM.
