2014 - Mazda6 - Engine

STARTER [SKYACTIV-G 2.5]

Purpose, Function

  • The starter rotates the crankshaft via the ring gear and it serves as a trigger to start the engine.

Construction

  • The starter is installed to the rear part of the engine on the left side.

  • The starter is a coaxial reduction type which can obtain high torque.

Operation

  • For starter cut-off control details, refer to CONTROL SYSTEM. (See STARTER CUT-OFF CONTROL [SKYACTIV-G 2.5].)

Fail-safe

  • Function not equipped.
