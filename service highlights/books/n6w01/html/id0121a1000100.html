2014 - Mazda6 - Engine

DRIVE SELECTION SYSTEM [SKYACTIV-G 2.5]

Outline

  • Switching between sport mode and normal mode is possible depending on how the drive selection switch is operated. When sport mode is selected, the operating force relative to the throttle opening angle is greater than normal mode.

Block Diagram

Operation

  • Switching between sport mode and normal mode is possible depending on how the drive selection switch is operated.

  • When normal mode is selected and the drive selection switch is pressed, the drive mode is switched to sport mode. When sport mode is selected and the drive selection switch is pressed, the drive mode is switched to normal mode.

  • When sport mode is selected, the operating force relative to the throttle opening angle is greater than normal mode.

  • When sport mode is selected, the PCM sends the following signals to each unit.

      • Drive selection signal

    • When the PCM sends a drive selection signal to the TCM, the TCM will change the shift points.

      Sport light illumination request signal

    • The PCM sends a sport light illumination request signal to the instrument cluster. The instrument cluster turns on the sport light to notify the driver that the drive selection control condition has been switched to sport mode.

Drive selection switch inhibition conditions

  • If any of the following conditions is met, the mode is not switched even if the drive selection switch is operated.

    • During sudden maneuvering

    • During DSC control

    • During ABS control

    • During TCS control

  • If the drive selection switch has been operated with a drive selection switching inhibition condition being met, the PCM sends a sport light flash request signal to the instrument cluster.

  • The instrument cluster flashes the sport lamp to notify the driver that drive selection is currently prohibited.

Drive selection stop conditions

  • If any of the following conditions is met while in sport mode, the drive mode is switched to normal mode.

    • Ignition is switched off

    • During cruise control operation

    • Malfunction with PCM control-related part
