2014 - Mazda6 - Suspension

TIRE PRESSURE MONITORING SYSTEM SET SWITCH

Purpose

  • The tire pressure monitoring system set switch is used to initialize the tire pressure monitoring system.

Construction, Operation

  • The tire pressure monitoring system set switch is installed to the dashboard.

  • The tire pressure monitoring system set switch has adopted a monitoring type switch.

  • The tire pressure monitoring system set switch sends a tire pressure monitoring system initialization signal to the instrument cluster.

Fail-safe



DTC

Fail-safe

C1031:35

  • Illuminates the tire pressure monitoring system warning light.

  • Inhibits the TPMS control.
