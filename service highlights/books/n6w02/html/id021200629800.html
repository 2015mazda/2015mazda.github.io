2014 - Mazda6 - Suspension

INSTANT MOBILITY SYSTEM

  • The instant mobility system enables temporary repair of a puncture without tire removal.

  • Directly install the bottle containing the repair agent to the air compressor to enable simultaneous charging of the repair agent and the air using compressed air from the compressor.

  • The instant mobility system is located in trunk compartment and includes the following:

    • Repair agent

    • Repair agent filler hose

    • Air compressor

    • Leaflet

    • Instruction manual

    • Speed limit label

    • Filled tire indication label

  • The accessory socket (12 V DC) is used as an input power source for the air compressor and the compressor plug includes a fuse.

NOTE:

  • The expiration date of the repair agent is printed on the repair agent bottle. Do not use the repair agent if the expiration date has expired.

  • Dispose of the repair agent according to local waste disposal law.
