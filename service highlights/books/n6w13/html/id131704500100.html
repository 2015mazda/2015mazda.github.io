2014 - Mazda6 - I-ELOOP

i-ELOOP WARNING ALARM [i-ELOOP]

Purpose

  • The i-ELOOP warning alarm is a function which warns the driver if the vehicle is driven before pre-charging is completed and while “i-ELOOP charging” is displayed in the LCD.

Function

  • The i-ELOOP warning alarm sound pattern is as shown in the figure.

Construction

  • The i-ELOOP warning alarm sounds using the buzzer built into the instrument cluster.

Operation

1. The instrument cluster receives (1) the i-ELOOP warning alarm request signal from the PCM.
2. The instrument cluster microcomputer turns the transistor on (2) based on the i-ELOOP warning alarm request signal.
3. When the transistor turns on, a ground circuit with the alarm is established and the alarm sounds (3).

Fail-safe

  • Function not equipped.
