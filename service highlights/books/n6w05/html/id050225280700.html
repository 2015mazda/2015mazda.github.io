2014 - Mazda6 - Transmission/Transaxle

ON-BOARD DIAGNOSTIC SYSTEM SIMULATION FUNCTION [FW6A-EL]

Purpose/Function

  • The simulation function can drive output parts based on use of the Mazda Modular Diagnostic System (M-MDS).

  • Inspection of an individual output part is facilitated by the simulation function because the output part is driven forcibly.
