2014 - Mazda6 - Transmission/Transaxle

AUTOMATIC TRANSAXLE SHIFT MECHANISM

Outline

  • A sport AT-type shift mechanism has been adopted for all models.

  • An electric shift-lock system has been adopted to prevent driver mis-operation.

  • An emergency manual shift-lock release system has been adopted to ensure emergency bypass if shifting operations cannot be performed.

Structural view

NOTE:

    There are two types of automatic transaxle shift mechanisms.

  • Type A: Gate-shaped key type. There is no push button on the selector lever knob.

  • Type B: Gate-shaped straight type. There is a push button on the selector lever knob.
