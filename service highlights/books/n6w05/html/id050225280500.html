ON-BOARD DIAGNOSTIC SYSTEM EXTERNAL TESTER COMMUNICATION FUNCTION [FW6A-EL]

Purpose/Function

  • The on-board diagnostic system external tester communication function transmits data between an external tester and each control module of the vehicle.

  • Because the DTCs recorded in the vehicle can be read by an external tester, determination of a malfunction location is facilitated.

Construction/Operation

  • A connector (DLC-2) conforming to International Organization for Standardization (ISO) standards has been adopted.

  • A connector shape and 16-pin terminal arrangement as stipulated by the ISO 15031-3 (SAE J1962) international standard has been adopted for this DLC-2.
