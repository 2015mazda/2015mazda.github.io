ON-BOARD DIAGNOSTIC SYSTEM PID/DATA MONITOR FUNCTION [FW6A-EL]

Purpose/Function

  • The PID/DATA monitor function can monitor control data such as input signals/internal calculation values/output signals using the Mazda Modular Diagnostic System (M-MDS).

  • The module control condition can be verified in real-time by the PID/DATA monitor function.
