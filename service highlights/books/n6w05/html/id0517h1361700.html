2014 - Mazda6 - Transmission/Transaxle

SHIFT POSITION INDICATOR LIGHT [FW6A-EL]

Purpose/Function

  • The shift position indicator light indicates the selector lever position.

Construction

  • The shift position indicator light is built into the instrument cluster.

Operation

  • The TCM sends a shift position indicator light illumination request signal to the instrument cluster via CAN based on the signal from the integrated transaxle range sensor signal received via CAN.
