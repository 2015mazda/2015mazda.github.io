2014 - Mazda6 - Multiplex Communication System

MULTIPLEX COMMUNICATION SYSTEM

  • A controller area network (CAN) system has been adopted in which multiplex communication is performed using a single communication path.

  • A high-speed controller area network (HS-CAN) has been adopted for the communication between the following modules:

    • PCM

    • DSC HU/CM

    • TCM (ATX)

    • AFS control module (with AFS system)

    • Front body control module (FBCM)

    • Vehicle control module (V/C-module) (with mazda radar cruise control (MRCC) system)

    • DC-DC converter (i-ELOOP) (with i-ELOOP system)

    • Position memory control module (with power seat)

    • Laser sensor (with smart city brake support system)

    • Forward sensing camera (with forward sensing camera)

    • EPS control module

    • Start stop unit

    • SAS control module

    • Audio amplifier (with Bose®)

    • Instrument cluster

  • A mid-speed controller area network (MS-CAN) has been adopted for the communication between the following modules:

    • Instrument cluster

    • Audio unit (with Bluetooth system)

    • SIRIUS satellite radio unit (with SIRIUS satellite radio)

    • Clock (with manual air conditioner)

    • Climate control unit

    • BSM control module (LH) (with BSM system)

    • Rear body control module (RBCM)
