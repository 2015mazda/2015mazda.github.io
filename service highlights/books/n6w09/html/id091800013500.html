2014 - Mazda6 - Body and Accessories

TURN SIGNAL/HAZARD WARNING INDICATOR LIGHTS

Purpose

  • Notifies the user that the turn lights are flashing.

Function

  • Flashes according to the turn switch and hazard warning switch operations.

Construction

  • Displayed in the instrument cluster.

Operation

Turn light system
1. When the turn switch is operated to the LH position, a turn switch LH signal is input to the start stop unit.
2. When the turn switch is operated to the RH position, a turn switch RH signal is input to the start stop unit.
3. The start stop unit sends the turn switch RH or LH signal to the front body control module (FBCM).
4. The front body control module (FBCM) sends the turn switch RH or LH signal to the instrument cluster as a turn signal indicator signal.
5. When a turn signal indicator signal is received, the instrument cluster flashes the turn signal indicator light.

Hazard warning system
1. When the hazard warning switch is turned on, a hazard warning signal is input to the start stop unit.
2. The start stop unit sends a hazard warning signal to the front body control module (FBCM).
3. The front body control module (FBCM) sends the hazard warning signal to the instrument cluster as a hazard warning indicator signal.
4. When the hazard warning indicator signal is received, the instrument cluster flashes the hazard warning indicator light.

Fail-safe

  • Function not equipped.
