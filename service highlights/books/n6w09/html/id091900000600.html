2014 - Mazda6 - Body and Accessories

WIPER/WASHER SYSTEM

Outline

  • The following wiper/washer system has been adopted.

    • Windshield wiper/washer system

    • Auto wiper system (with auto wiper system)

    • Washer fluid-level sensor (with washer fluid-level sensor)

Specification

  • Windshield wiper/washer system with continuous (low/high), auto-stop, one-touch wiper, intermittent wiper, and synchronized washer and wiper operation adopted.

    • CAN communication has been adopted for the windshield wiper/washer system.

    • Windshield washer nozzles have been adopted in which the area where the washer fluid is sprayed can be adjusted.

  • An auto wiper system has been adopted which enables fully automatic windshield wiper operation.

    • CAN communication has been adopted for the auto wiper system.

    • A rain sensor integrated with the lens has been adopted which aggregates infrared light to detect the amount of rainfall.

    • Auto wiper system personalization features have been adopted.

  • A washer fluid-level sensor is installed to the washer tank.

  • Washer tank capacity

    • With washer fluid-level sensor: 4.2 L {4.4 US qt, 3.7 Imp qt}

    • Without washer fluid-level sensor: 2.2 L {2.3 US qt, 1.9 Imp qt}
