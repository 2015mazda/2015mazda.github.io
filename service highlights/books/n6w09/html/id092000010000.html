2014 - Mazda6 - Body and Accessories

ENTERTAINMENT SYSTEM

Outline

  • The following entertainment system has been adopted.

    • Audio system

    • Car-navigation system (with car-navigation system)

    • Park assist system (with park assist system)

    • Bluetooth system (with Bluetooth system)

  • An audio unit with integrated AM/FM tuner/MP3/WMA (Windows Media Audio)/AAC (Advanced Audio Coding) applicable CD player in the audio panel.

  • Steering switch has been adopted.

  • Commander switch has been adopted. (with commander switch)

  • Auxiliary jack/USB port has been adopted.

  • An antenna amplifier has been adopted.

  • A SIRIUS satellite radio antenna has been adopted. (with SIRIUS satellite radio system)

  • A glass antenna has been adopted.

  • A rear mount camera has been adopted.
