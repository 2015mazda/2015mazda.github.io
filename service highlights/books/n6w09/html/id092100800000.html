MAIN FUSE

Purpose

  • If electric current of the specified value or more flows to each fuse, the fuse melts due to heat generation to protect the systems and parts downstream of the fuse.

Function

  • If electric current to each fuse is the specified value or more, the fuse melts due to heat generation.

Construction

  • Installed to the battery terminal (+).

    Without i-ELOOP

    With i-ELOOP

Operation

1. If electric current to each fuse is the specified value or more, the fuse melts due to heat generation.
2. Current between the fuse and ground is interrupted.

Fail-safe

  • Function not equipped.
