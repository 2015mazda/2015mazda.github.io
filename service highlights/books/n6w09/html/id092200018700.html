DOOR-AJAR WARNING LIGHT

Purpose

  • The door-ajar warning light notifies the driver that any door or the trunk lid is open.

Function

  • When the instrument cluster receives any of the following signals sent from the rear body control module (RBCM) via the CAN signal, the door-ajar warning light illuminates.

    • Door latch switch ON signal (door open)

    • Trunk lid latch switch ON signal (trunk lid open)

Construction

  • The instrument cluster controls illumination on/off of the door-ajar warning light using the instrument cluster microcomputer based on the door latch or trunk lid latch position signal from the rear body control module (RBCM).

  • The door-ajar warning light is set in the instrument cluster.

  • The door-ajar warning light in the type A instrument cluster is displayed in the LCD. Refer to the DOOR AJAR WARNING DISPLAY for the display content.

Operation

1. The instrument cluster receives (1) the door latch switch ON signal or trunk lid latch switch ON signal from the rear body control module (RBCM).
2. The instrument cluster microcomputer turns the transistor on (2) based on each signal.
3. When the transistor turns on, a ground circuit with the door-ajar warning light is established and the door-ajar warning light illuminates (3).

Fail-safe

  • Function not equipped.
