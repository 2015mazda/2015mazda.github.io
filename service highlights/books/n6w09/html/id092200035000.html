FOW WARNING ALARM

Purpose

  • The FOW warning alarm warns the user of a possible collision with a vehicle ahead.

Function

  • If the instrument cluster receives a warning sound request signal from the vehicle control module (V/C-Module), is sounds the FOW warning alarm using the buzzer in the instrument cluster.

  • The FOW warning alarm sound pattern is as indicated in the figure.

Construction

  • The FOW warning alarm sounds using the buzzer built into the instrument cluster.

Operation

1. The instrument cluster receives (1) the warning sound request signal from the vehicle control module (V/C-Module).
2. The instrument cluster turns the transistor on (2) based on the warning sound request signal.
3. When the transistor turns on, a ground circuit with the alarm is established and the alarm sounds (3).

Fail-safe

  • Function not equipped.
