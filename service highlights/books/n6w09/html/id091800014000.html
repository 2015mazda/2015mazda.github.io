AFS OFF INDICATOR LIGHT

Purpose

  • The AFS OFF indicator light notifies the user that the adaptive front lighting system (AFS) is stopped or a malfunction occurs in the adaptive front lighting system (AFS).

Function

  • The AFS OFF indicator light illuminates when the adaptive front lighting system (AFS) or the headlight auto leveling system is stopped and flashes when a malfunction occurs in the adaptive front lighting system (AFS) or the headlight auto leveling system.

Construction

  • Displayed in the instrument cluster.

Operation

NOTE:

  • When the ignition is switched ON (engine off or on), the AFS control module illuminates the AFS OFF indicator light for 3 s to check the light bulb and notifies that the AFS OFF indicator light bulb is normal.

  • The AFS control module sends a AFS OFF indicator light illumination/flash request signal to the instrument cluster when a malfunction occurs in the adaptive front lighting system (AFS) or the headlight auto leveling system is stopped.

  • The instrument cluster illuminates/flashes the AFS OFF indicator light based on the AFS OFF indicator light illumination/flash request signal.

Fail-safe

  • Function not equipped.
