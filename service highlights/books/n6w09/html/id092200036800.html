MAINTENANCE MONITOR

Outline

  • When the preset maintenance period approaches, the maintenance monitor displays a message on the LCD display in the instrument cluster to inform the driver.

Function/Construction

  • The maintenance monitor has the following items.

  • Maintenance settings for the scheduled maintenance, tire rotation, and tire pressure and maintenance display/non-display can be set according to user option. Refer to the Workshop Manual for the setting procedure.

  • The instrument cluster stores the maintenance monitor setting content, and the elapsed time and traveled distance from when the setting was made. The stored content is not cleared even if the battery is removed.

NOTE:

  • Because the maintenance monitor setting information stored in the instrument cluster cannot be written to a new instrument cluster, if the instrument cluster is replaced, the maintenance monitor is the initial set value. Before replacing the instrument cluster, explain to the customer that the maintenance monitor is reset to the initial setting.

Operation

  • If any of the following display conditions is met during the preset maintenance period, the maintenance monitor displays a message.

      • Display conditions

    • Day-setting: Remaining days are 15 days or less

    • Distance setting: Remaining distance is 500 km/250 mile

  • The instrument cluster displays a message in which the display conditions are met when 0.8 s have elapsed after the ignition is switched ON (engine off or on).

  • If several display conditions are met, the instrument cluster displays a message at approx. 5 s intervals according to the indication order of precedence (See maintenance monitor table).

  • When an INFO button operation signal on the steering switch is received via CAN signal from the start stop unit while a message is displayed, the instrument cluster clears messages in which 2 s or more have elapsed since the message was displayed.

    NOTE:

    • The cleared message can be verified on the warning message verification screen again.
