ROOM LIGHT CONTROL SYSTEM

Outline

  • The room light control system changes the illumination condition and illumination level of the room lights (door position) according to whether the doors are opened/closed and locked/unlocked.

  • The room light control system is controlled by the rear body control module (RBCM).

  • The rear body control module (RBCM) performs room light control system fail-safe. (See REAR BODY CONTROL MODULE (RBCM).)
