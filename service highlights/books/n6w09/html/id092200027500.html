BLIND SPOT MONITORING (BSM) OFF INDICATOR LIGHT

Purpose

  • The BSM OFF indicator light informs the driver that the Blind Spot Monitoring (BSM) system is turned off or there is a malfunction in the system.

Function

  • If the instrument cluster constantly receives a BSM OFF indicator signal via CAN signal from the blind spot monitoring (BSM) control module (LH), it illuminates the BSM OFF indicator light.

  • If the instrument cluster periodically receives a BSM OFF indicator signal via CAN signal from the blind spot monitoring (BSM) control module (LH), it flashes the BSM OFF indicator light.

  • The flash pattern of the BSM OFF indicator light is as shown in the figure.

Construction

  • The BSM OFF indicator light are equipped in the instrument cluster.

Operation

  • The instrument cluster receives (1) the BSM OFF indicator signal from the BSM control module (LH) via the CAN signal.

  • The instrument cluster turns the transistor on (2) intermittently when the BSM OFF indicator signal is periodically received and continuously when the signal is constantly received.

  • When the transistor turns on intermittently, the BSM OFF indicator light flashes (3) and when it turns on continuously, the BSM OFF indicator light illuminates (3).

Fail-safe

  • Function not equipped.
