2014 - Mazda6 - Body and Accessories

KEYLESS RECEIVER

Purpose, Function

  • Receives ID data from the remote transmitter and sends the received ID data to the start stop unit.

Construction

  • Installed inside the tire house trim.

Operation

1. The ID data sent from the remote transmitter is received by the keyless receiver.
2. The keyless receiver transmits the received ID data to the start stop unit.

Fail-safe

  • Function not equipped.
