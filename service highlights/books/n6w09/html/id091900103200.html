WIPER/WASHER SYSTEM PERSONALIZATION

  • A personalization function has been adopted which switches the auto wiper system between operable/inoperable.

Using Mazda Modular Diagnostic System (M-MDS)

  • The auto wiper function can be enabled or disabled using the Mazda Modular Diagnostic System (M-MDS). Refer to the Workshop Manual for the personalization function setting procedure.
