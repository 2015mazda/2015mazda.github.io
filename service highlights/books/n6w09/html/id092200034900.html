FORWARD OBSTRUCTION WARNING (FOW) SYSTEM

Outline

  • The Forward Obstruction Warning (FOW) system detects the distance to a vehicle ahead or an obstruction using the radar sensor, and if the vehicle control module (V/C-Module) determines that there is the danger of a collision with a vehicle ahead or obstruction, the system alerts the driver of the possible collision using the warning indication and warning sound.

WARNING:

  • If the accelerator pedal or brake pedal is mistakenly operated, it could result in an accident. Always verify the safety of the surrounding area and depress the brake pedal while keeping a safe distance from the vehicle ahead.

  • The FOW system may not operate under the following conditions:

    • If there is the possibility of hitting only a part of a vehicle or obstruction ahead.

    • If the vehicle is accelerated rapidly and it comes close to a vehicle ahead.

    • The DSC HU/CM is malfunctioning.

    • The vehicle is driven at the same speed as the vehicle ahead.

    • The accelerator pedal is depressed.

    • The brake pedal is depressed.

    • The steering wheel is being operated.

    • The selector lever is being operated.

    • The turn signal is being used.

  • If the radar sensor aiming adjustment for the FOW system is not completed, a deviation in the radar emission range may occur which could result in an accident. When performing the following servicing, always perform the radar sensor aiming adjustment. Refer to the radar sensor aiming adjustment procedure in the workshop manual.

    • Vehicle control module (V/C-module) removal/installation, replacement

    • Radar sensor, radar sensor bracket removal/installation, replacement

CAUTION:

  • Tires for all four wheels must always be of the specified type and size. If tires other than the specified type and size are used, the FOW system may not operate normally.

  • Do not mix tire types or use tires with significantly different wear patterns on the same vehicle. Use of inappropriate tires could result in the FOW system not operating normally.

  • Do not tow the vehicle pointed backward with driving wheels on the ground. This may cause internal damage to the transaxle.

  • Towing a trailer with this vehicle is dangerous because it has not been designed to tow a trailer and doing so will affect the drive system which could result in vehicle damage.

NOTE:

  • The FOW system does not operate at speeds under 15 km/h{9.3 mph}, or while the vehicle is reversing.

  • Although the system is designed to recognize 4-wheeled vehicles, the radar sensor may recognize the following types of vehicles or objects as a vehicle ahead or obstruction, and the FOW system could operate.

    • Objects close to the entrance of a curve.

    • An on-coming vehicle appears while the vehicle is cornering to the left.

    • Narrow railing on a bridge, narrow gate or entrance.

    • Near the boundary of a slope and flat road surface such as at the entrance to an underground parking area.

    • Speed restriction bumps, projections, and reflective metallic objects set up on the road.

    • Tall plants and high-standing grass and forage

    • Plastic curtain or flag shaped objects

    • 2-wheeled vehicles (motorcycles, bicycles).

    • Pedestrians, animals and trees.

Function

  • The FOW system uses the vehicle control module (V/C-Module) to calculate the distance and relative speed of a vehicle ahead or obstruction based on the vehicle-ahead or obstruction information sent from the radar sensor, and if it determines that there is the danger of a collision (time until collision with vehicle ahead or obstruction decreases below threshold), the system warns of the danger of a collision using the warning screen and warning sound.

Personalization features

  • The following FOW system settings can be changed: Refer to the workshop manual for details on the personalization setting method.

    • System ON (operational)/OFF (non-operational) (initial set value is ON)

    • Operation timing for FOW warning indication and warning sound (initial set value is Near)

    • FOW warning alarm volume (initial set value is High)

System status display function

  • The vehicle control module (V/C-Module) displays the system status in the instrument cluster LCD display.

Operation

Operation conditions

  • The operation is performed when all of the following conditions are met:

    • FOW system ON (operational)

    • Vehicle speed calculated by vehicle control module (V/C-Module), based on wheel speed signals received via CAN communication from DSC HU/CM, is 15 km/h {9.3 mph} or more

    • Relative speed of obstruction ahead calculated by vehicle control module (V/C-Module) is approx. 15—145 km/h {9.4—90.0 mph}

    • Vehicle control module (V/C-Module) determines danger of collision with obstruction ahead (time until collision with obstruction ahead decreases below threshold)

Operation

  • When the vehicle control module (V/C-Module) determines that there is the danger of a collision with an obstruction ahead (time until collision decreases below threshold) based on the relative speed and distance of the obstruction ahead, it sends a warning indication request signal and warning sound request signal to the instrument cluster.

  • When the instrument cluster receives the warning indication request signal and warning sound request signal from the vehicle control module (V/C-Module), it displays the warning screen in the LCD and sounds the FOW warning alarm using the buzzer.
