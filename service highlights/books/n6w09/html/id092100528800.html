2014 - Mazda6 - Body and Accessories

FUSE BLOCK

Purpose

  • Fuses used by each system are grouped and housed together for smooth inspection and replacement servicing.

Function

  • Fuses used by each system are grouped and housed together.

Construction

  • Installed inside the front side trim (LH).

Operation

Fuse operation
1. If electric current to each fuse is the specified value or more, the fuse melts due to heat generation.
2. Current between the fuse and ground is interrupted.

Fail-safe

  • Function not equipped.
