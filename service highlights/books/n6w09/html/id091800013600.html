HEADLIGHT HIGH-BEAM INDICATOR LIGHT

Purpose

  • Notifies the user that the headlights HI are illuminated.

Function

  • Illuminates when the headlight HI or passing illumination conditions are met.

Construction

  • Displayed in the instrument cluster.

Operation

1. When the light switch is operated to the HI or FLASH-TO-PASS position, a light switch HI or FLASH-TO-PASS signal is input to the start stop unit.
2. The start stop unit sends the light switch HI or FLASH-TO-PASS signal to the front body control module (FBCM).
3. The front body control module (FBCM) sends the light switch HI or FLASH-TO-PASS signal to the instrument cluster as a headlight HI illumination signal.
4. When the instrument cluster receives the headlight HI illumination signal, it illuminates the headlight high-beam indicator light.

Fail-safe

  • Function not equipped.
