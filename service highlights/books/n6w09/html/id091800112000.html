STEERING ANGLE SENSOR

Purpose, Function

  • The steering angle sensor detects the steering angle and sends it to the start stop unit as a steering angle (absolute steering angle) signal.

NOTE:

  • The steering angle sensor is equipped if any of the following systems is equipped.

    • Adaptive front lighting system (AFS)

    • Smart city brake support (SCBS)

    • Blind sport monitoring (BSM) system

Construction

  • The steering angle sensor is installed to the clock spring.

  • Consists of a sensor unit in which a photo transistor and LED are positioned opposite each other, and a slit disc which moves together with the steering wheel.

Operation

  • As the disc moves with the steering wheel operation, the LED light received by the photo transistor varies due to the slits on the disc.

  • The photo transistor reads the variation of the light and outputs the steering angle (absolute steering angle) signal to the start stop unit.

Fail-safe

  • Function not equipped.
