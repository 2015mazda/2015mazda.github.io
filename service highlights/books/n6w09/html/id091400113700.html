TRUNK LID OPENER SYSTEM

Outline

  • A trunk lid opener system has been adopted in which the trunk lid can be opened just by pressing the trunk lid opener switch.

  • The rear body control module (RBCM) performs the trunk lid opener system fail-safe. (See REAR BODY CONTROL MODULE (RBCM).)

Function

  • When the trunk lid opener switch is pressed, the trunk lid release actuator operates to open the trunk lid.

  • The trunk lid opener system is controlled based on the following CAN signal:



CAN signal sending module

Signal name

Via instrument cluster from PCM

Vehicle speed signal

Structural view

System wiring diagram

Operation

Operation condition

When all of the following conditions are met, the trunk lid opener system operates.

  • Vehicle speed is 3 km/h {2 mph} or less

Trunk lid opener switch (Interior) open operation
1. When the trunk lid opener switch is pressed, it turns on (1).
2. When the operation condition of the trunk lid opener system is met, the rear body control module (RBCM) turns the transistor on for approx. 0.7 s (2) and turns on the trunk lid release relay.
3. When the trunk lid release relay turns on, the trunk lid release actuator operates (3).
4. When the trunk lid release actuator operates, the latch lever is freed and the trunk lid lifts up simultaneously via the spring force of the trunk lid weatherstrip and stopper rubber, and opens.

Trunk lid opener switch (exterior) open operation
1. When the trunk lid opener switch is pressed, it turns on (1).
2. When the operation condition of the trunk lid opener system is met, the rear body control module (RBCM) turns the transistor on for approx. 0.7 s (2) and turns on the trunk lid release relay.
3. When the trunk lid release relay turns on, the trunk lid release actuator operates (3).
4. When the trunk lid release actuator operates, the latch lever is freed and the trunk lid lifts up simultaneously via the spring force of the trunk lid weatherstrip and stopper rubber, and opens.
