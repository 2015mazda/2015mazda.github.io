2014 - Mazda6 - Body and Accessories

REAR MAP LIGHT

Purpose

  • The rear map lights shine light at the center of the vehicle interior.

Function

  • The rear map lights illuminate when the rear map light switch is turned on.

  • The rear map lights illuminate according to the lock condition of the driver's door and open/close conditions of the doors under the rear body control module (RBCM) control when the room light switch for front map light is in the DOOR position.

Construction

  • The rear map lights are assembled to the headliner.

Operation

Rear map light switch operation
1. The rear map light switch is on.
2. The battery current flows to the rear map light bulb from the rear body control module (RBCM).
3. The rear map light illuminates.

Room light switch for front map light operation
1. Room light switch for the front map light is in the DOOR position.
2. When any door is opened, the door open signal is input to the rear body control module (RBCM).
3. When the rear body control module (RBCM) receives the door open signal, it turns on the transistor.
4. When the transistor turns on, the battery current flows to the rear map light bulbs from the rear body control module (RBCM).
5. The rear map lights illuminate.

NOTE:

  • When the room light switch is in the DOOR position, the rear map light illumination on/off timing is controlled by the room light control system. For details, refer to the "ROOM LIGHT CONTROL SYSTEM".

Fail-safe

  • Function not equipped.
