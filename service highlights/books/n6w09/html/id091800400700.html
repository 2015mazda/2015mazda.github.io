ADAPTIVE FRONT LIGHTING SYSTEM (AFS) CONTROL MODULE

Purpose

  • The AFS control module calculates the target optical axis based on the vehicle height signal and outputs the target optical axis signal to the headlight leveling actuator.

  • The AFS control module outputs the target headlight optical axis signal to the swivel actuator according to the steering wheel operation while the headlights are turned on.

Function

  • The AFS control module performs control of the adaptive front lighting system (AFS) and headlight auto leveling.

Automatic configuration function

  • When the ignition is switched ON (engine off or on) after the AFS control module is replaced, the AFS control module reads the vehicle specification information sent via CAN communication from the instrument cluster, and stores the vehicle specification information.

On-board diagnostic function

  • The AFS control module is equipped with an on-board diagnosis function which records DTCs in the event of a malfunction.

    Malfunction detection function

  • Detects malfunctions of the adaptive front lighting system (AFS) and the headlight auto leveling system related parts.

    Display Function

  • If any malfunction is detected, the AFS OFF indicator light in the instrument cluster flashes to notify the driver of a system malfunction.

Operation

  • For the AFS control module operation, refer to "HEADLIGHT AUTO LEVELING SYSTEM" and "ADAPTIVE FRONT LIGHTING SYSTEM (AFS)".
