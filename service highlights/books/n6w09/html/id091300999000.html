POSITION MEMORY SYSTEM

Outline

  • A position memory system has been adopted in which the position of the driver’s seat can be programmed and positioned.

  • The position memory system is controlled by the position memory control module.

Function

Seat position detection function

  • The position memory control function can detect the seat position by the pulse signals it receives from the position sensors integrated with each motor.

  • The position memory control module receives the pulse signals from the position sensors integrated with each motor when the power seat switch or the position memory switch is operated.

Seat position programming function

  • The position memory control module can program seat positions corresponding to each individual remote transmitter or to seat positions corresponding to each switch of the position memory switch.

  • The position memory control module can program seat positions using the remote transmitter or the position memory switch when all of the following conditions are met.

    • Remote transmitter is used

      • Ignition is switched OFF (LOCK)

      • Power seat system is not operating

      • Seat positioning function is not operating

    • Position memory switch is used

      • Power seat system is not operating

      • Seat positioning function is not operating

Seat positioning function

  • The position memory control module can automatically position the seats programmed to each individual remote transmitter or the seat positions programmed to each switch of the position memory switch.

  • The position memory control module can automatically position programmed seat positions using the remote transmitter or the position memory switch when all of the following conditions are met.

    • Remote transmitter is used

      • Ignition is switched OFF (LOCK)

      • Power seat system is not operating

      • Seat positioning function is not operating

      • Seat position is programmed using remote transmitter

      • Press the unlock button of the remote transmitter and open the driver’s door within 40 s.

    • Position memory switch is used

      • Power seat system is not operating

      • Seat positioning function is not operating

      • Seat position is programmed using position memory switch

      • Vehicle speed is 3 km/h or lower (ignition switched ON (engine off or on))

      • ATX: Shift lever is in P position (ignition switched ON (engine off or on))

      • MTX: Parking brake is applied (ignition switched ON (engine off or on))

Programmed seat position clearing function

  • The position memory control module can clear all seat positions programmed using the position memory switch and remote transmitters when any of the following conditions is met.

    • Remote transmitter is programmed or replaced, or cleared.

    • Negative battery cable is disconnected.

    • ROOM 15 A fuse is removed

    • If any position sensor is malfunctioning, the position memory control module operates the fail-safe function.

  • The position memory control module can clear seat positions programmed to individual remote transmitters when the following operations are performed using the remote transmitter.

    NOTE:

    • Clearing of programmed seat positions can be performed while the power seat system and seat positioning functions are not operating.

    • Ignition is switched OFF (LOCK).

    • The position memory switch SET switch is pressed.

    • Within approx. 5 s from Step 2, the remote transmitter lock button is pressed.

Operation

Seat position programming

  • Using remote transmitter

    • Operate the power seat switch to move the seat to the position you want to program.

    • When the position memory switch SET switch is pressed, the SET switch on signal is input to the position memory control module.

    • When the position memory control module receives the SET switch on signal, it transitions the system to the seat position programming mode and sounds the buzzer.

    • When the remote transmitter unlock button is pressed within 5 s from Step 3, the start stop unit sends ID data and an unlock signal to the position memory control module.

    • When the position memory control module receives the ID data and unlock signal, it records the seat position, returns to the normal mode, and sounds the buzzer.

  • Position memory switch is pressed (Ex.: Memory switch 1 used).

    • Operate the power seat switch to move the seat to the position you want to program.

    • When the SET switch of the position memory switch is pressed, a SET switch on signal is input to the position memory control module.

    • When the position memory control module receives the SET switch on signal, it transitions the system to the seat position programming mode and sounds the buzzer.

    • When memory switch 1 is pressed within 5 s from Step 3, a memory switch 1 signal is input to the position memory control module.

    • When the position memory control module receives the memory switch 1 signal, it records the seat position, returns to the normal mode, and sounds the buzzer.

Seat positioning

  • Remote transmitter is used

    • If an unlock operation is performed using the remote transmitter or a front door request switch, the start stop unit sends ID data and an unlock signal to the position memory control module.

    • When the driver’s door is unlocked within 40 s from Step 1, the instrument cluster sends a driver’s door front door open signal to the position memory control module.

    • When the position memory control module receives the ID data and unlock signal, and the driver’s door front door unlock signal, it starts the seat positioning operation.

    • When the position memory control module starts the seat positioning operation, is calculates the difference between the current seat position and the programmed seat position and determines the direction of movement and distance for each motor.

    • The position memory control module supplies current to each motor based on the determined direction of movement and distance.

    • When the position memory control module detects that the seat has moved to the programmed position, it stops the seat positioning operation and sounds the buzzer.

  • Position memory switch is used (Ex.: Memory switch 1 is used)

    • When memory switch 1 is pressed, the memory switch 1 signal is input to the position memory control module.

    • When the position memory control module receives the memory switch 1 signal, it starts the seat positioning operation.

    • When the position memory control module starts the seat positioning operation, is calculates the difference between the current seat position and the programmed seat positions and determines the direction of movement and distance for each motor.

    • The position memory control module supplies current to each motor based on the determined direction of movement and distance.

    • When the position memory control module detects that the seat has moved to the programmed position, it stops the seat positioning operation and sounds the buzzer.
