POWER WINDOW SUBSWITCH

Purpose

  • Each door glass except for the driver-side can be opened/closed with occupant seated in the cabin using the power window subswitch.

Function

Manual switch

  • The power window subswitch switches the contact point according to the switch operation and switches the circuit to the power window motor.

Auto switch

  • When the power window subswitch is operated for auto open/close operation, it sends an auto open/close signal to the P/W CM.

Construction

  • The power window subswitch is equipped on the front passenger side and the rear passenger sides.

Operation

Manual operation (With driver's side window auto open/close) (Example: close operation)
1. When the power window subswitch is turned to the manual close position (1), a manual close signal is sent to the power window motor.(2)

Manual operation (With all window auto open/close) (Example: close operation)
1. When the power window subswitch is operated for manual close operation, the power window motor (LF) starts manual close operation.(1)
2. When the P/W CM receives a manual close signal, it turns on the close relay. (2)
3. When the close relay is turned on, the power window motor (RF) starts manual close operation.(3)

Auto operation (With all window auto open/close) (Example: close operation)
1. When the power window subswitch is turned to the auto close position (1), it sends an auto close signal to the CPU.
2. When the CPU receives an auto close signal, it turns on the close relay.(2)
3. When the close relay is turned on, the power window motor (LR) starts close operation.(3)

Fail-safe

  • Function not equipped.
