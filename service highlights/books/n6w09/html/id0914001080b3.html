KEYLESS ANTENNA [KEYLESS ENTRY SYSTEM]

Purpose

  • Outputs a request signal and specifies the remote transmitter location.

Function

  • The keyless antennas output request signals based on the signals from the start stop unit.

Construction, Operation

  • Request signals are output by the keyless antenna installed in the rear console to produce the keyless antenna output range inside the vehicle.

Fail-safe

  • Function not equipped.
