PUSH BUTTON START

Purpose

  • By pressing the switch ON/OFF, the push button start notifies the start stop unit of the user power supply switching operation.

Function

START/STOP/ENGINE indicator light

  • The characters in the push button start illuminate to notify the user of the button position.

Construction

  • The push button start is installed in the meter hood.

  • Two types of switches are integrated in the push button start.

  • The push button start has the following integrated indicators. For the indicator inspection conditions, refer to the POWER SUPPLY SWITCHING.

    • ACC indicator

    • ON indicator

    • Indicator (green)

    • Indicator (amber)

  • The push button start has an integrated coil antenna for the immobilizer system. (See COIL ANTENNA.)

Operation

  • When the push button start is pressed, switches 1 and 2 turn on (contact point: closed), and when released, it turns off (contact point: open).

START/STOP/ENGINE indicator light

  • When a door is open/closed, the START/STOP/ENGINE indicator light illuminates when the headlights are turned on.

Fail-safe

  • Function not equipped.
